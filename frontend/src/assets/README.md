# Assets Directory

## Ragdoll Cat Logo

Пожалуйста, добавьте файл `ragdoll-cat.jpg` в эту папку.

Файл должен содержать фотографию кота Ragdoll, которая будет использоваться как логотип в интерфейсе.

Поддерживаемые форматы: .jpg, .jpeg, .png, .webp

Если вы используете другое имя файла или формат, обновите импорты в:
- `frontend/src/components/Layout.jsx`
- `frontend/src/pages/RagdollPage.jsx`
